# OMRR Simulator

The Simulator is used to send constant messages to the Logserver. Currently it supports 
* DCC
* MFX

## Parameters

| short | long     | description                     |
|-------|----------|---------------------------------|
| -r    | --random | randomize time between messages |
| -n    | --name   | hostname for the simulator      |
